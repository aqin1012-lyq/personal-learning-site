# GitHub Release Template

## Title

v0.1.0 - First public-ready template release

## Summary

This release turns the project from a personal learning site into a reusable personal learning system template.

It now gives users a clearer path to:
- understand the product positioning
- create logs, notes, and projects
- use `/studio` as an onboarding and publishing surface
- share the site with stronger metadata and launch assets

## Highlights

- stronger homepage and README positioning
- template onboarding and empty states
- scaffolding scripts for content creation
- `/studio` support for logs, notes, and projects
- `gray-matter` based content parsing
- content validation for required fields and date formats
- default OG image, favicon, and launch assets

## Recommended next action for users

1. edit `src/data/site.ts`
2. replace the default about/current-learning content
3. publish your first log, note, and project
4. deploy to Vercel and set the real domain

## Notes

If you're using this as a real public site, make sure to replace placeholder text and the example domain before launch.
