## Long version

A lot of people do learn consistently, but their learning system is fragmented.

The process is in journals.
The concepts are scattered across notes.
The output lives in GitHub or random folders.

So I built a personal learning system template.

Instead of treating learning like a pile of disconnected posts, it organizes it into:
- Logs for learning process
- Notes for reusable knowledge
- Projects for real output

The goal is not to look busy.
The goal is to make learning easier to revisit, connect, and continue.

If you’ve ever felt like you learned a lot but didn’t really retain a system, this template is for you.
